# Changelog

## [1.0.0] - 2026-06-09

### Added
- Initial release
- Customer management (CRUD with search, sort, pagination)
- Product/service catalog (CRUD with search, sort, pagination)
- Invoice creation with PDF export (CRUD with search, filter, sort, pagination)
- Quote creation with PDF export (CRUD with search, filter, sort, pagination)
- Dashboard with statistics (total customers, products, invoices, quotes)
- Company profile settings (company name, tax info, address, logo, currency)
- Tax/VAT rate settings per invoice/quote
- Multi-currency support
- Payment status tracking (draft, pending, paid, overdue, cancelled)
- Quote status tracking (draft, sent, accepted, rejected)
- Search and filtering across all list pages
- Column sorting on list pages
- Date range filtering on invoices and quotes
- Docker support with MySQL and Redis
- Demo data seeder (admin user, sample customers, products, invoices, quotes)
